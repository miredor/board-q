# quest-board

Небольшой CLI-проект на Rust в стиле fantasy board с квестами.

## Что умеет

- генерировать случайные квесты
- добавлять свои квесты вручную
- смотреть список
- отмечать квест выполненным
- смотреть статистику
- хранить данные локально в файле `quests.db`

## Запуск

```bash
cargo run -- generate 3
cargo run -- add "Помыть кружку архимага"
cargo run -- list
cargo run -- done 2
cargo run -- stats
```

## Команды

### Сгенерировать квесты

```bash
cargo run -- generate 5
```

### Добавить свой квест

```bash
cargo run -- add "Разобрать стол"
```

### Показать все квесты

```bash
cargo run -- list
```

### Завершить квест

```bash
cargo run -- done 1
```

### Статистика

```bash
cargo run -- stats
```

## Хранение данных

Все квесты сохраняются в файл `quests.db` рядом с исполняемым файлом / текущей директорией запуска.

## Почему без зависимостей

Проект специально написан без внешних crate, чтобы он был максимально простым и легко читаемым.
